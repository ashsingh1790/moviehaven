# Domain


## Documents

* [Film](film.md) - The core domain object — a movie with TMDB metadata, cast, crew, and scores.
* [List item](list-item.md) - A film's placement within a list, with ordering and an optional note.
* [List](list.md) - A user-curated collection of films (watchlist or custom list).
* [Rating](rating.md) - A user's score for a film, optionally carrying review text.
* [User](user.md) - A registered account that owns ratings and lists.
